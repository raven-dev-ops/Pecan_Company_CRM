# Discovery Decision: MVP Workflow and Screens

Issue link: #4

## Minimum screens
1. Settings
2. Products
3. Customers
4. Ring-Up
5. Tender/Finalize
6. Sales History

## End-to-end MVP workflow
1. Open app and verify connection/receipt folder settings.
2. Optionally select customer during checkout.
3. Add products to cart (unit quantity or weight).
4. Choose payment method.
5. Finalize sale (persist sale + sale items in one transaction).
6. Generate local PDF receipt and store predictable filename.
7. View/reprint from Sales History.

## Non-goals for MVP
- No live payment processor integration.
- No card authorization/capture flow.
- Payment method is recorded only (Cash/Card/Other).
